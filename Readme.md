# Homepage of Junwon Lee

after commit, deploy

```
git commit origin main
yarn deploy
```

### Setup
0. (Optional) Make conda environment
  ```bash
  conda create -n homepage
  conda activate homepage
  ```
1. Install nodejs
   ```bash
   conda install -c conda-forge nodejs=18.14.1
   ```
2. Install yarn
   ```bash
   npm install -g yarn@1.22.21
   ```
3. Install modules
   ```bash
   yarn install
   ```
4. ```bash yarn init```
